# 📊 Sales Analysis Dashboard (SQL + Power BI)

---

## 🎯 Objective

Analyze sales data to identify top-performing customers, products, and regions, and generate actionable business insights using SQL and Power BI.

---

## 🗃 Dataset Description

The dataset contains transactional sales data with the following fields:

| Column        | Description             |
| ------------- | ----------------------- |
| order_id      | Unique order identifier |
| customer_name | Name of the customer    |
| region        | Customer region         |
| product       | Product purchased       |
| amount        | Order value             |
| date          | Transaction date        |

---

## ⚙️ Data Processing (SQL)

Performed analysis using SQL:

* Total revenue calculation
* Revenue by region
* Revenue by product
* Top customers by spending
* Identifying highest-performing region and product

---

## 📊 Dashboard Overview

The Power BI dashboard includes:

* 💰 Total Revenue KPI
* 🌍 Revenue by Region
* 📦 Revenue by Product
* 👤 Top Customers by Revenue
* 📋 Sales Detail Table
* 🎛 Interactive Filters (Product, Region)

---

## 🔍 Key Insights

* East region generates the highest revenue
* Laptop is the top-performing product
* A small group of customers (Ayesha, Sara, Ali) contributes most of the revenue
* Several customers show low or no activity

---

## 💡 Business Recommendations

* Retain high-value customers through loyalty strategies
* Improve engagement for low-activity customers
* Focus marketing efforts on high-performing products and regions
